# Homebrew Formulae

```
brew tap mcjkula/tap
```

## Available

| Formula | Install |
|---------|---------|
| [taolk](https://github.com/mcjkula/taolk) | `brew install mcjkula/tap/taolk` |

## How it works

Formulae are updated automatically when a new release is tagged. The source repo's release workflow computes SHA256 hashes for each binary and dispatches an update to this tap.
